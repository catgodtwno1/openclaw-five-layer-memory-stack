# MemOS

## Role

MemOS is the external lifecycle/shared memory layer.

## Why it is separate

MemOS is not trying to replace local retrieval or chat compaction.
Instead, it adds:

- external service-based memory storage
- lifecycle-oriented management
- future multi-machine / multi-agent reuse

## Local validation target

A minimal successful validation includes:

- service reachable
- add endpoint works
- search endpoint works
